## Features

* Adds an "IINA" button to YouTube video pages
* Opens videos at the current playback position
* Seamless one-click transition

## Prerequisites

* macOS
* [IINA](https://iina.io/) installed
* [Tampermonkey](https://www.tampermonkey.net/) browser extension installed

## Installation

### 1. Add the Shortcut

[Click here to install the Shortcut](https://www.icloud.com/shortcuts/215f4b4d01e94560ac25568b21751388)

### 2. Add the Userscript

With Tampermonkey installed, [click here](https://raw.githubusercontent.com/sammrai/iina.shortcut/main/iina.shortcut.user.js) for automatic installation

## Usage

1. Open a YouTube video
2. Click the "IINA" button below the video player
3. IINA will automatically launch and start playing from the current position

## Troubleshooting

If YouTube videos won't open in IINA, see the [Troubleshooting Guide](TROUBLESHOOTING.md).
